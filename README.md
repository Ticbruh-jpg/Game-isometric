# Game-isometric

A vibe-coded isometric game inspired by Vampire Survivors, built with Godot 4.

## About

Top-down isometric action game where you survive waves of enemies. Vibe-coded — built for fun, not polish.

## Requirements

- [Godot 4](https://godotengine.org/download/)

## Running

1. Clone the repo
2. Open Godot 4
3. Import the project (`project.godot`)
4. Hit Play
